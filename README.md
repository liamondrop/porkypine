porkypine
=========

A CoffeeScript implementation of some classic data structures

#### `SinglyLinkedList([dataItems])`
 * `sll.at(position)`
 * `sll.push(dataItem)`
 * `sll.insertAfter(position, dataItem)`
 * `sll.insertBeginning(dataItem)`
 * `sll.removeAfter(position)`
 * `sll.removeBeginning()`
 * `sll.toArray()`
 * `sll.toString()`

#### `Queue()` [FIFO]
